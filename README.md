# TelegramDotaStatsBot 🎮📊

Telegram-бот для получения актуальной статистики по **Dota 2** прямо в мессенджере.  
Позволяет быстро узнать информацию об игроке, матче, героях и популярных сборках.

## Возможности 🚀

- 👤 **Статистика игрока** — рейтинг, винрейт, и другая информация по `dota_id`.
- ⚔️ **Статистика матча** — детали конкретной игры (длительность, победитель и другая информация).
- 🗡️ **Статистика героев** — общий винрейт героя, популярность, средние показатели.
- 🛠️ **Сборки на героев** — актуальные билды с высоким винрейтом из текущего патча.
